# Codex 目前最爆火的 Skills（2026-03）

> 说明：Codex 官方没有公开“下载量排行榜”。这份清单按当前社区讨论热度和实用性整理。

## 1. skill-installer（安装技能）

**为什么爆火**
- 这是新用户上手最快的入口：一条流程就能把现成技能装进 `$CODEX_HOME/skills`。
- 支持从精选列表安装，也支持从 GitHub 仓库路径安装，扩展性强。
- 团队协作时复用效率很高，能快速统一工作流。

**适合场景**
- 想快速把某类能力（如测试、发布、审查）接入 Codex。
- 多项目复用同一套技能模板。
- 需要从外部仓库同步技能能力。

## 2. skill-creator（创建技能）

**为什么爆火**
- 当通用技能不够用时，它能把你的个人/团队方法论沉淀成可复用能力。
- 对复杂流程（多步骤命令、规范、模板）特别友好。
- 能显著减少重复提示词和重复操作。

**适合场景**
- 把团队 SOP 封装成标准化技能。
- 为特定技术栈（前端、后端、数据、DevOps）定制工作流。
- 持续迭代“私有技能库”，提升交付一致性。

## 怎么选

- **先快用**：优先 `skill-installer`，先拿现成技能跑起来。
- **再深定制**：用 `skill-creator` 把高频流程固化成你的专属技能。
- 最佳实践通常是：**installer 打底 + creator 精修**。
